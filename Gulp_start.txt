// Что-бы стартонуть новый проэкт 
1. Копируем старт папку
2. Переиминовуем
3. Открываю проэкт в VsCode
4. Открываю терминал в VsCode
5. >npm i
6. Потгружаются все зависимости.
7. НЕ ЗАБЫВАЙ формат файлов должен быть в UTF-8
8. Для запуска Gulp >gulp
9. Для завершения проэкта и создания папки dist
   >gite build

10. Для работы с Git >git init
